# Contributing to bmap.ai

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Write tests
5. Submit a pull request

## Code Style
- Follow PEP 8
- Use type hints
- Write docstrings

## Testing
Run all tests before submitting:
```bash
python scripts/run_tests.py
```

## Commit Messages
Use clear, descriptive commit messages.
